###179. 最大数. 给定一组非负整数，重新排列它们的顺序使之组成一个最大的整数。
#### 思路
使用比较器，将所有的数字转换为字符串进行比较，但需要将每次比较都将两个数先组合再交换
位置组合，选择其中较大值的排序为准。如果给数组最大值为0，直接返回0即可
#### 代码
```java
class Solution {
    public String largestNumber(int[] nums) {
        String[] strnum=new String[nums.length];
        for(int i=0;i<nums.length;i++){
            strnum[i]=String.valueOf(nums[i]);
        }

        Arrays.sort(strnum,new Comparator<String>(){
	        public int compare(String s1,String s2){
		        String ss1=s1+s2;
                String ss2=s2+s1;
                return ss2.compareTo(ss1);
	        }
        });

        if(strnum[0].equals("0")){
            return "0";
        }
        String str = new String();
        for(int i=0;i<strnum.length;i++){
            str+=strnum[i];
        }
        return str;
    }
}
```

###342. 4的幂：编写一个函数，其作用是将输入的字符串反转过来。输入字符串以字符数组
 char[] 的形式给出。不要给另外的数组分配额外的空间，你必须原地修改输入数组、使用
 O(1) 的额外空间解决这一问题。你可以假设数组中的所有字符都是 ASCII 码表中的可打印字符。
#### 思路
由于该题要求在原数组上进行修改，所以需要先得出对应数字翻转后的位置，然后进行替换即可
#### 代码
```java
class Solution {
    public void reverseString(char[] s) {
        int len=s.length;
        char temp;
        for(int i=0;i<len/2;i++){
            temp=s[i];
            s[i]=s[len-1-i];
            s[len-1-i]=temp;
        }
    }
}
```
